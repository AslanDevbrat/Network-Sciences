Show transcribed image text Consider an undirected (connected) tree of n vertices. Suppose that a particular vertex in the tree has degree k, so that its removal would divide the tree into k disjoint regions, and suppose that the sizes of those regions are n,, n2,n3, .nk. Show that the unormalized betweeness centrality x of the vertex is 2

Answer:

Betweenness Centrality of a node n is the ratio of the shortest paths between all other nodes, that pass through node n.

This can be expressed as difference between number of shortest paths between any 2 vertices in the tree and the shortest path between any 2 vertices in the disjointed regions-n1,n2,n3,....nk

= n*n - (n1*n1 + n2*n2 + n3*n3 +.... + nk*nk)

= n^2 - Sigma(1,k) ni^2
